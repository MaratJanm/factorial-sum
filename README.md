# factorial-sum

Небольшой учебный проект на Python: вычисляет факториал суммы двух чисел `a` и `b`.

Файлы:
- `main.py` — пример с переменными `a` и `b`, выводит `factorial(a + b)`.

Запуск:
```
python main.py
```

Описание:
- Подойдёт для изучения базовых операций с числами и написания простых функций.
